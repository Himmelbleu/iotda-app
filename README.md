# 项目介绍

调用华为云 iotda 提供的 API，制作一个 py 客户端管理设备和获取设备数据，实现简单的功能。

## 目录介绍

- iotda-client：python 客户端，操作华为云设备 API、操作数据库。
- iotda-serve：java 服务器，面向于 web，操作数据库提供接口。
- iotda-web：vue3 项目，渲染数据库数据。
- database：数据库 sql 文件。
